# MTGA — MAKE TEXT GREAT AGAIN

Плагин для Claude Code, который заставляет Claude отвечать в стиле твитов Дональда Трампа. КАПС. Восклицания. Самовосхваление. В каждом ответе в чате — а не только в коммитах.

> Лучший плагин в истории плагинов. Все так говорят.

🇬🇧 [English README](README.en.md)

Вдохновлён [MowoGroup/MCGA](https://github.com/MowoGroup/MCGA) (Make Commits Great Again), который делает это для сообщений коммитов. MTGA распространяет идею на всё общение — а коммиты остаются отдельным тумблером, выключенным по умолчанию.

---

## Что внутри

1. **Output styles** (`MTGA` — русский, `MTGA-EN` — английский) — Claude отвечает, отчитывается и объясняет в стиле трамповских твитов, при этом пути файлов, номера строк, результаты тестов и честность остаются нетронутыми. Стиль применяется ТОЛЬКО к тексту в чате: код, файлы, документация и команды не трогаются никогда.
2. **100 спин-фраз** (паки RU + EN) — строчки статуса внизу превращаются в «Построю стену из тестов — баги заплатят» и «У меня есть концепция плана».
3. **Тумблеры на всё** — стиль, спины и трамп-коммиты включаются и выключаются независимо. Твои прежние `outputStyle` и `spinnerVerbs` бэкапятся и восстанавливаются.

**Обычный Claude:**

> Я исправил проверку токена в auth/middleware.js — она выполнялась после редиректа. Все 14 тестов проходят.

## Установка

```
/plugin marketplace add howdeploy/MTGA
/plugin install mtga@mtga
```

Дальше:

```
/mtga:on        # русский стиль + спины
/mtga:on en     # английский стиль + спины
```

## Команды

| Команда | Что делает |
| --- | --- |
| `/mtga:on [ru\|en]` | Включить стиль ответов + спин-фразы (прежние настройки бэкапятся) |
| `/mtga:off` | Выключить всё, восстановить прежние `outputStyle` и `spinnerVerbs` |
| `/mtga:status` | Показать, что включено: стиль, язык, коммиты, спины |
| `/mtga:commits on\|off` | Трамп-стиль для **сообщений коммитов** — отдельный тумблер, по умолчанию выключен |
| `/mtga:spins on\|off [ru\|en]` | Только спин-фразы, независимо от стиля ответов |

Стиль и спины вступают в силу после `/clear` или новой сессии (Claude Code читает их на старте). Флаг коммитов работает сразу.

## Как это устроено

- Стиль ответов — обычный [output style](https://docs.claude.com/en/docs/claude-code/output-styles) Claude Code, лежит в `output-styles/`. `/mtga:on` прописывает `outputStyle` в `~/.claude/settings.json`.
- Спины пишутся в настройку `spinnerVerbs` (`mode: replace`). Прежние фразы сохраняются в `~/.claude/mtga-state.json` и восстанавливаются при `off`.
- Тумблер коммитов — флаг в `~/.claude/mtga-state.json`; output style инструктирует Claude проверять его перед написанием любого сообщения коммита. `false` → обычный conventional commit, без исключений.

## Требования

- Claude Code с поддержкой плагинов
- `bash` + `python3` (Linux, macOS; на Windows — WSL или Git Bash)

## Удаление

Выполни `/mtga:off` **до** удаления плагина — он восстановит исходные `outputStyle` и `spinnerVerbs`. Если забыл: убери ключи `outputStyle` и `spinnerVerbs` из `~/.claude/settings.json` (или восстанови их из `~/.claude/mtga-state.json`) и удали `~/.claude/mtga-state.json`.
